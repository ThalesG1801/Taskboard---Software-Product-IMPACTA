# TaskBoard

**Descrição:**  
TaskBoard é um sistema de gerenciamento de tarefas inspirado em quadros tipo Trello. O projeto permite criar, listar e deletar tarefas organizadas em colunas, oferecendo uma visão simples e funcional para controle de atividades.

---

## Estrutura do projeto
taskboard/  
├── Backend → Código do servidor Node.js + Express  
├── Frontend → Interface visual em HTML, CSS e JavaScript  
└── database → Banco de dados SQLite
